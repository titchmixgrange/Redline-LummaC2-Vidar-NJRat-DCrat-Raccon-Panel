Description:

This project is a controlled educational lab environment designed to study common behavioral patterns of remote access tools (RATs) and data-exfiltration threats in a safe and isolated setup. It includes simulated modules that replicate typical techniques used in real-world scenarios without containing any harmful or deployable code.



Purpose:



Study of cybersecurity threats in a controlled environment



Understanding behavior patterns of remote access mechanisms



Training in detection and mitigation techniques



Academic research and security awareness



System Requirements:



Windows 10/11 or Linux (recommended virtual machine)



8 GB RAM minimum



Virtualization software (e.g., VirtualBox or VMware)



Isolated test environment
